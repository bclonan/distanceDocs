## How to use

### Developer Widget

**Directions**

Go to the project directory in your consol. 

```sh
cd D:pathtprojectdirectory/timeWidget
```

Install project dependencies

```sh
npm install
```

Run locally

```sh
npm run dev
```

Or Build

```sh
npm run build
```

Alternatively you can use vue cli 3

[Vue cli](https://cli.vuejs.org/guide/)
